# Google Gemini AI Chatbot

A simple and professional command-line AI chatbot built using Python and the official Google Gemini API (`gemini-2.5-flash`).

## Features
- Real-time conversational AI using Google Gemini.
- Error handling and clean user prompt structure.
- Built using professional coding guidelines.

## How to Run
1. Clone the repository.
2. Install dependencies: `pip install google-genai`
3. Set your Gemini API Key in `bot.py`.
4. Run the bot: `python bot.py`
